# PYTHON Basic Grammar
---
## 类型
* 整数
  | 类型 | 符号 | 符号 |
  | --- | --- | ---|
  | 二进制 | 0B | 0b |
  | 八进制 | 0O | 0o |
  | 十六进制 | 0X | 0x |
* 浮点数 *不确定尾数*
* 复数 z.real//实部  z.imag//虚部

## 字符串
[m,n,k] //从m到n步进为k
## 函数
pow(x,y,z) //$x^{y}$ mod z;
